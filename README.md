# 📌 Stack Implementation in C 

# 📖 Overview 

This project implements a fixed-capacity stack data structure in C. 

The stack supports push, pop, and peek operations and includes explicit overflow and underflow protection.

The goal was to practice:

- Static memory allocation
- Setting and keeping track of a stack-pointer. 
- Boundary checking (Overflow and Underflow).
- Memory layout awareness. 

# 🧰 Build

**OS:** Linux Ubuntu 24.04.4 LTS
**Code Editor:** VIM 
**Compiler:** gcc 13.3.0 
**Debugger:** GNU gdb 15.0.50
